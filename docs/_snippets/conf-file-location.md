The [recommended installation](../infrastructure/installation/index.md) uses the config file `/etc/xrpld/xrpld.cfg` by default. Other places you can put a config file include `$HOME/.config/ripple/xrpld.cfg` (where `$HOME` is the home directory of the user running `xrpld`), `$HOME/.local/ripple/xrpld.cfg`, or the current working directory from where you start `xrpld`.
