# Escrow (JavaScript)

Demonstrates how to create, finish, and cancel escrows on the XRP Ledger.

## Setup

```sh
npm install
```

## Send Timed Escrow

```sh
node send-timed-escrow.js
```

## Send Conditional Escrow

```sh
node send-conditional-escrow.js
```

## List Escrows

```sh
node list-escrows.js
```

## Cancel Escrow

```sh
node cancel-escrow.js
```
