# Checks (JavaScript)

Demonstrates how to create, cash, and cancel checks on the XRP Ledger.

## Setup

```sh
npm install
```

## Create a Check

```sh
node create-check.js
```

## Cash a Check

```sh
node cash-check-exact.js
```

```sh
node cash-check-flexible.js
```

## Cancel a Check

```sh
node cancel-check.js
```

## Get Checks

```sh
node get-checks.js
```
